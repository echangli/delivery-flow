## Политика

- Работаем только с **dev/stage**; **прод не мутируем** (в Ф7 — только read-baseline, без записи).
- Мутации тест-данных на стейдже — можно, **только заявленными фикстурами** + журнал + откат (Ф9).
- Скилл **не пушит код, не деплоит, не мержит** (как весь delivery-флоу). Клон репо — read-only.
- Деструктив (drop/flush/delete/`<оркестратор> delete·apply`) — никогда.

- **URL слота фронта:** ⟪ADAPT: паттерн URL слота вашего стейджа⟫.
- **Бэкенд API-хост / gateway на стейдже:** ⟪ADAPT: хост API/шлюза стейджа; ws-хост (если есть WS-канал)⟫.
- **VPN:** ⟪ADAPT: нужен ли VPN до стейджа; обычно включён у пользователя⟫.

  Бан-страница/403 на edge = egress-IP не в вайтлисте (сообщи пользователю IP).

## Infra-гейты (заранее недостижимое — ⛔, циклы не трать)

  байпасится. Внешние `X-Forwarded-For`/`X-Real-IP` для geoip обычно игнорируются.
- Реальные платёжные провайдеры / SMS / внешние OAuth — только на ручную (Ф3, не-агентские).
